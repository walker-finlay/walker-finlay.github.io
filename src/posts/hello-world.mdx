### Testing out features

#### Basics
- markdown rendering
- internal links: treated as a re-render rather than a redirection
- external links 
- footers [^1]
#### Content Delimiters
Clicking the html themed content delimiters will scroll you to the top or bottom of the page.
#### LaTeX
Renders math with [`katex`]() via [`gatsby-remark-katex`](https://www.gatsbyjs.com/plugins/gatsby-remark-katex/?=gatsby-remark-) 
in inline mode e.g. $\sum_{n=1}^\infty\frac{1}{n}$ or display mode e.g.

$$
P(A\vert B)=\frac{P(B\vert A)P(A)}{P(B)}
$$
#### React components 
React clock from [`wojtekmak/react-clock`](https://github.com/wojtekmaj/react-clock) directly in a markdown file. 

<LiveClock />

#### Syntax Highlighting
Uses [`prism`](https://prismjs.com/) via [`gatsby-remark-prismjs`](https://www.gatsbyjs.com/plugins/gatsby-remark-prismjs/) slightly (prompt is unselectable) customized for interactive python
```ipython{numberLines: true}
>>> sq = (x**2 for x in range(10))
>>> list(sq)
[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
>>> 
```
